Aug. 18 2019
For Industry hosted Capstone project
Portfolio Optimizer

This Project is a Python Dash based web application named Portfolio Optimizer/Portfolio Allocator. 
It provides a graphical user interface that allows users to make choices and upload own csv file to get a optimally allocated portfolio.
The main function is inside the app.py file. It imports, ie. uses functions from DataPreparation.py, DataPreparation_BL.py, Optimizer, Resampled
BL_optimizer.py, sort_version.py and a folder named pypfopt.
test.csv and BL_test.csv are examples we created for our project.
